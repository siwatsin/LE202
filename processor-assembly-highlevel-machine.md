# ภาษาชั้นสูง - high level language
เป็นภาษาชั้นสูงที่พัฒนามาจากภาษาเครื่องและภาษาแอสเซ็มบลี้ ซึ่งมีความสะดวกมากในการเขียนและแ่านโปรแกรมได้ง่ายขึ้น ใช้งานโดยแปลจากภาษาชั้นสูงเป็นภาษาเครื่อง แต่มีข้อเสียตรงที่สามารถใช้ได้แค่บนคอมพิวเตอร์ชนิดเดียวกันเท่านั้น

# โปรเซสเซอร์ - processor,microcontroller
โปรเซสเซอร์ เป็นระบบคอมพิวเตอร์ที่มีหน้าที่ในการคำนวณและประมวลผล ใช้หน่วยความจำและคำสั่งจากอุปกรณ์อื่นๆ

ไมโครคอนโทรลเลอร์ เป็นอุปกรณ์ที่ประกอบไปด้วยชุดข้อมูล หน่วยความจำ คำสั่ง เพื่อนำไปเชื่อมต่อกับระบบให้เกิดการทำงานตามที่ต้องการ

# ภาษาแอสเซ็มบลี้ - low level language, assembly
เป็นภาษาที่พัฒนามาจากภาษาเครื่อง มีการใช้ตัวอักษรภาษาอังกฤษแทนเลขฐานสองในการทำงาน เข้าใจได้ง่ายและสามารถสั่งงานได้รวดเร็ว แต่มีซับซ้อนในเรื่องการคำนวณ จึงไม่นิยมใช้

# ภาษาเครื่อง - machine language
เป็นภาษาที่มีระดับต่ำที่สุด การเขียนโปรแกรมแทนด้วยเลขฐานสองทั้งหมด คอมพิวเตอร์สามารถอ่านได้ทันทีและรวดเร็ว แต่ไม่เป็นที่นิยมเนื่องจากทำความเข้าใจได้ยาก

# ความสัมพันธ์ระหว่างภาษาต่างๆ
เริ่มแรก ภาษาเครื่องเป็นภาษาที่่คอมพิวเตอร์เข้าใจได้ทันทีแต่การเขียนโปรแกรมเป็นเรื่องที่เข้าใจได้ยาก จึงได้มีการพัฒนาภาษาต่อมาคือ ภาษาแอสเซ็มบลี้ โดยใช้ตัวอักษรภาษาอังกฤษแทนเลขฐานสอง ซึ่งทำให้เข้าใจได้ง่ายขึ้น แต่ยากในการคำนวณ เลยมีการพัฒนาเพื่อให้เข้าใจได้ง่ายขึ้นไปอีก จึงได้มีภาษาชั้นสูงขึ้นมาซึงสะดวกและง่ายต่อการเข้าใจ แต่เนื่องจากคอมพิวเตอร์เข้าใจภาษาเครื่องเพียงเท่านั้นจึงจะสามารถทำงานตามคำสั่งได้ เลยต้องมีการแปลเป็นภาษาเครื่องในท้ายที่สุด

# ความสัมพันธ์ระหว่างภาษากับโปรเซสเซอร์
ภาษาเป็นเหมือนคำพูดที่ใช้ในการสื่อสารกับคอมพิวเตอร์ โดยการเขียนโปรแกรมส่งบนคอมพิวเตอร์ โปรเซสเซอร์จะทำหน้าที่แปลคำสั่งต่าง ๆ คำนวณและประมวลผล เพื่อนำข้อมูลและหน่วยความจำไปใส่ไว้ในไมโครคอนโทรลเลอร์เพื่อใช้ในการทำงานตามคำสั่ง
